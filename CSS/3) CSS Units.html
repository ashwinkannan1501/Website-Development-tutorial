Units in CSS

Absolute Units :-

  1. The text will be styled in pixel
  2. The text will be styled in point
  3. The text will be styled in picas
  4. The text will be styled in centimenter
  5. The text will be styled in millimeter

Relative Units :-

  • The text will be displayed in Percentage
  • The text will be displayed in Ephermeral Unit
  • The text will be displayed in Root Epheremeral Unit
  • The text will be displayed in Viewport Height
  • The text will be displayed in Viewport Width
  • The text will be displayed in Character Unit
  • The text will be displayed in EX
  • The text will be displayed in ViewPort Minimum
  • The text will be displayed in ViewPort Maximum
